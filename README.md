# CredBridge

### Building Financial Credibility Before Credit History

CredBridge is a student financial credibility platform that helps students build a verified Financial Passport, generate a TrustScore™, and unlock personalized opportunities before they have a traditional credit history.

---

## The Problem

Millions of students are excluded from valuable opportunities because they lack:

* Credit history
* Income history
* Loan repayment records
* Traditional financial identity

As a result, students struggle to access:

* Scholarships
* Student discounts
* Software benefits
* Startup programs
* Educational financing opportunities
* Banking products

Traditional systems evaluate financial history.

Students often have none.

---

## Our Solution

CredBridge creates a verified Financial Passport that captures a student's achievements, skills, experiences, leadership activities, and financial readiness.

Using this profile, CredBridge generates a TrustScore™ and recommends opportunities tailored to the student's profile.

Instead of asking:

> "What has this student borrowed before?"

CredBridge asks:

> "How ready and credible is this student today?"

---

## Key Features

### Financial Passport

A verified student profile containing:

* Academic achievements
* Certifications
* Projects & portfolios
* Internships & experience
* Leadership activities
* Financial literacy indicators

---

### TrustScore™

A student credibility and opportunity-readiness score.

**Range:** 300–900

TrustScore helps students:

* Track progress
* Understand growth areas
* Unlock future opportunities
* Improve financial readiness

---

### Verification Engine

Every claim is assigned a confidence level.

Verification sources include:

* Self-reported information
* Document uploads
* OCR validation
* GitHub verification
* LinkedIn verification
* Institutional verification (future)

Outputs:

* Verification Confidence
* Verified Evidence Percentage
* Verification Badges

---

### Opportunity Marketplace

Students receive personalized recommendations for:

* Scholarships
* Student discounts
* Software benefits
* Competitions
* Startup programs
* Internship opportunities

---

### Opportunity Roadmap

Students can see:

* Current opportunities
* Locked opportunities
* Requirements needed to unlock them

This transforms TrustScore into a growth journey rather than a static number.

---

### Score Simulator

Students can simulate profile improvements and instantly see:

* Projected TrustScore
* New opportunities unlocked
* Recommended next actions

---

### AI Trust Coach

An AI-powered assistant that helps students:

* Improve their TrustScore
* Understand missing requirements
* Discover relevant opportunities
* Build stronger profiles

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/credbridge.git
cd credbridge
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
OPENAI_API_KEY=
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
